# Hostel

Bienvenido/a al proyecto **Hostel**. Este repositorio contiene el código fuente de una web de presentación para un hostal, desarrollada usando HTML, CSS y JS básicos.

## Descripción

La aplicacion web está diseñada para mostrar la información de un hostal de forma clara, atractiva y moderna. Con una estructura sencilla y estilos responsivos, los visitantes pueden conocer las características del alojamiento, ver imágenes, consultar servicios y encontrar información de contacto.

## Características

- Información de ubicación.
- Diseño responsive para dispositivos móviles y de escritorio.
- Operaciones de registro de cliente/reserva/habitación.


## Estructura del proyecto

- `/index.html` — Página principal
- `/estilos.css/` — Carpeta de archivos de estilos CSS


## Cómo usar

1. Clona este repositorio:
   ```bash
   git clone https://github.com/KevinWired/Hostel.git
   ```
2. Abre el archivo `index.html` en tu navegador para ver la web localmente.
3. Puedes modificar los archivos HTML y CSS para personalizar la web según tus necesidades.
